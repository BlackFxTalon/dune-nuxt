# Dune Imperium Fansite

Nuxt-based fan site with Nuxt UI + Nuxt Content and a repository-first data layer.

## Setup

```bash
corepack pnpm install
```

## Scripts

```bash
corepack pnpm dev
corepack pnpm build
corepack pnpm preview
corepack pnpm typecheck
corepack pnpm lint
corepack pnpm test:unit
corepack pnpm test:e2e
```
